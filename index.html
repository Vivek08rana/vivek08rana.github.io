York University - Grant Clinic
Automation Website

Flask | Python | HTML | CSS | JavaScript

Created and maintained a website using the Flask framework to streamline administrative tasks for the Grant Clinic hosted by York University's Research Commons.

Website Features:
(1) Managing email templates, allowing users to seamlessly view email templates and modify them with input fields.
(2) Automating the peer review email process by utilizing Pandas Python library to read an Excel spreadsheet and Docx Python library to read a template Word file, generating over 100 emails tailored to individual peer reviewers.
(3) Modifying the default email templates that are stored in the website.
